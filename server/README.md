# Tempguard server

This is the server for Tempguard. It is in charge of talking between the web/apps/whatever and the Dragonchain L1.

Current endpoints are:

* /alerts
* /alerts/:alert_id
* /equipment
* /equipment/:equipment_id
* /equipment/:equipment_id/history
* /rules
* /rules/:rule_id
* /sensors
* /sensors/:sensor_id
